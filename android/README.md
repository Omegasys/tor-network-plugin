# TOR Network Plugin for Android

This Android app routes all network traffic through the TOR network using the VPNService API. It ensures that your device connects to the internet via TOR, improving anonymity and privacy.

## Features

- Routes all network traffic through TOR
- Simple start/stop buttons for enabling and disabling the VPN connection
- Lightweight integration with the Tor Android service

## Setup

### Prerequisites

- Android Studio (latest version recommended)
- Android device or emulator running Android 5.0 (Lollipop) or higher

### Installing the App

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/tor-network-plugin.git
   Open the project in Android Studio.
Build and run the app on an emulator or physical device.
Grant the necessary permissions for network access and VPN service when prompted.
Usage
Launch the app on your device.
Click Start VPN to connect to the TOR network.
Click Stop VPN to disconnect from the TOR network.
Troubleshooting

If the app fails to start the VPN or connect to the TOR network, ensure that you have the necessary permissions enabled and that the device has an active internet connection.
