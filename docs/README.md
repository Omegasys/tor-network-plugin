# TOR Network Plugin

This project is a cross-platform plugin that routes network traffic through the TOR network. The plugin supports multiple platforms, including **Linux**, **macOS**, **Windows**, and **Android**, and it allows users to easily control and configure their system's TOR connection.

## Features

- **Cross-platform support**: Compatible with Linux, macOS, Windows, and Android.
- **Simple UI**: Provides an easy-to-use interface for starting and stopping the TOR service.
- **Security**: Routes all system traffic through the TOR network to ensure privacy.
- **Logging**: Provides logging utilities for tracking TOR connection status and troubleshooting.
