# Installation Guide

Follow the steps below to install the TOR network plugin on various operating systems.

## Prerequisites

Before installing the plugin, make sure you have **TOR** installed on your system. You can download it from the [TOR Project](https://www.torproject.org/download/).

Additionally, the plugin may require administrative privileges to control system network settings.

---

## Installation on Linux

### Step 1: Install TOR

If TOR is not already installed, you can install it using your package manager.

For **Debian/Ubuntu**:
```bash
sudo apt update
sudo apt install tor
