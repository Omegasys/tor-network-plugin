# TOR Network Plugin for iOS

This iOS app routes all network traffic through the TOR network using the **Network Extension** framework. It allows you to set up a VPN connection that ensures all internet traffic is anonymized through TOR.

## Features

- Routes all network traffic through TOR
- Simple start/stop buttons for enabling and disabling the VPN connection
- Lightweight integration with the Tor iOS service

## Setup

### Prerequisites

- Xcode (latest version)
- An iOS device running iOS 12.0 or higher (VPN services require physical devices, not simulators)
- CocoaPods for managing dependencies
